Receipt React Frontend

React frontend for the Receipt Project. Provides a user interface for uploading receipts, viewing saved receipts, and interacting with the Java backend.

Features

- Upload receipts to be processed by the Python OCR service
- View and filter saved receipts
- Interactive dashboard with receipt statistics
- JWT-based authentication with login and registration
- Responsive layout with sidebar and navbar

Tech Stack

- React 18
- React Router – page navigation
- CSS Modules – component-specific styling
- Fetch API – communication with backend services
- Vite – build tool and development server
